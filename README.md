# UD3 PSOC

This is the firmware for the UD3 Teslacoil driver. It is originally developed by Steve Ward. This is a fork with changed programming.

* FreeRTOS based
* CLI over USB or UART
* Internal MIDI interrupter 
* Autotuning function
